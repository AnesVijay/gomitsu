## "Защищённый канал" (через chatID)
- люди начинают диалог, мне приходит уведомление, разрешать ли им общаться с ботом
- если я разрешаю, чел записывается в БД

## "Письма"
- отправка только мне новых заявок
- отправка людям (я как ответ кидаю ссылку, она отправляется только тому человеку, на чьё сообщение в боте я "ответил")

## Оплата
- хранить статистику, кто когда оплатил
- напоминание об оплате vpn
    (приходит за неделю до даты конца доступа)

## БД
- таблица Account (хранит ник, имя профиля, данная пользователю ссылка подключения, chatID = уникальный id)
- таблица Payment (хранит дату оплаты, дату отключения доступа, сумму, ник = ключ)
